# Coolify Deployment Guides

Coolify is a self-hosted alternative to Heroku, Vercel, and Netlify. It uses Docker to deploy your applications with minimal configuration.

## Available Guides

- [Deploy to Coolify](/files/devops/coolify/deploy-to-coolify) - Complete guide to deploying your application
- [Auto Deploy with Git Push](/files/devops/coolify/auto-deploy-setup) - Set up automatic deployments

## What is Coolify?

Coolify allows you to:
- Deploy applications with Docker
- Auto-deploy on Git push
- Manage multiple applications
- Set up custom domains with SSL
- Configure environment variables
- Monitor application logs

## Quick Start

1. Set up Coolify (self-hosted or cloud)
2. Connect your Git repository
3. Configure your application
4. Deploy!

See our [Deploy to Coolify guide](/files/devops/coolify/deploy-to-coolify) for detailed instructions.
